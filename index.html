Spring Boot 常用註解速查表

適合後端工程師分階段學習，含 API、資料庫、驗證與安全性應用。

1. 應用啟動與配置類

註解用途
@SpringBootApplication啟動主類（含組件掃描、配置、自動設定）
@Configuration用來建立設定類別
@Bean註冊自定義 Bean
@Value從 application.yml/properties 注入變數

2. 控制器與 API 路由

註解用途
@RestController建立 REST API 控制器
@RequestMapping, @GetMapping, @PostMapping 等設定 API 路由
@RequestParam讀取 URL 查詢參數
@PathVariable讀取 URL 中的路徑變數
@RequestBody接收 JSON 格式的 request body
@ResponseBody回傳 JSON 結果（內含於 @RestController）

3. 服務層與依賴注入

註解用途
@Service標記服務層類別
@Autowired自動注入元件
@Component泛用元件標記

4. 資料庫與 DAO 層（JPA）

註解用途
@Entity對應資料表的 Entity 類
@Table(name="...")指定資料表名稱
@Id主鍵標記
@GeneratedValue自動生成主鍵策略
@Column欄位對應
@Repository標記 DAO 類別
@Transactional交易控制，確保資料一致性

5. 驗證與例外處理

註解用途
@Valid啟用參數驗證
@NotNull, @Size, @Email 等常見欄位驗證註解
@ControllerAdvice全域例外處理配置
@ExceptionHandler自訂例外處理方法

6. 安全性與授權

註解用途
@EnableWebSecurity啟用 Spring Security
@PreAuthorize("hasRole('ROLE_ADMIN')")方法級權限驗證
@Secured("ROLE_USER")基於角色的存取控制

建議學習順序

優先級重點註解說明
⭐ 必學@SpringBootApplication, @RestController, @GetMapping, @Service, @Autowired基本 CRUD 與 API 應用
👍 進階@RequestBody, @Valid, @Transactional, @Repository資料處理與驗證
🔐 安全性@PreAuthorize, @ControllerAdvice登入驗證與錯誤控制
